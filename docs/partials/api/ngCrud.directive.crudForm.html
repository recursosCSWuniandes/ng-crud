crudForm
directive in module ngCrud

Description

creates form inputs based on a fields description

Usage

as element:
<crud-form
       fields="{object}"
       record="{object}"
       list-of-values="{object}">
</crud-form>

Directive info

  • This directive creates new scope.
  • This directive executes at priority level 0.

Parameters

ParamTypeDetails
fieldsobject

description of fields

recordobject

object in which values will be set

listOfValuesobject

key/value objects where key is an attribute name and values are lists of values

Example

<crud-form fields="fields" record="myRecord" lists-of-values="lovs"></crud-form>
