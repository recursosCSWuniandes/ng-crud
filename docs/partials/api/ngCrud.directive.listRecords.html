listRecords
directive in module ngCrud

Description

Creates a table showing the registered fields for every record in records

Usage

as element:
<list-records
       records="{array}"
       fields="{object}"
       actions="{object}"
       [checklist="{boolean}"]>
</list-records>

Directive info

  • This directive creates new scope.
  • This directive executes at priority level 0.

Parameters

ParamTypeDetails
recordsarray

Array with records to display

fieldsobject

definition of the fields

actionsobject

Actions available per record

checklist
(optional)
boolean

Whether or not to show checkboxes
