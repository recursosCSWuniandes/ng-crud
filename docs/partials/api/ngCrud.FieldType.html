FieldType
service in module ngCrud

Description

Defines the configuration for an object attribute

Properties

  • displayName

    Defines the text to show on labels

    Returns
    string

    Display name for the field

  • required

    Sets the field as required

    Returns
    boolean

    Whether or not the field is required

  • type

    Defines the type of the field. Currently the valid types are:

    • String
    • Image
    • Boolean
    • Reference
    • Date
    • Currency
    • Computed
    • Long
    • Number
    • Integer
    Returns
    string

    Name for field's type

Example

{
    displayName: 'String' //Name to display in forms or lists,
    type: '', //Type of the field
    required: false //Whether the field es required or not
}
