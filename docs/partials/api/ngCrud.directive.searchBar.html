searchBar
directive in module ngCrud

Description

Creates a search form

Usage

as element:
<search-bar
       name="{expression}"
       fields="{object}"
       record="{expression}"
       submit-fn="{function}">
</search-bar>

Directive info

  • This directive creates new scope.
  • This directive executes at priority level 0.

Parameters

ParamTypeDetails
nameexpression

name to show in toolbar

fieldsobject

definition of the search fields

recordexpression

object to which the result is mapped

submitFnfunction

function to execute when submitting
